Website portofolio statis dengan tampilan modern, navigasi responsif (hamburger menu), dark mode, dan section profil singkat.

## Preview Section
Website berisi beberapa section:
- Home
- About
- Skills (card bisa di-flip saat diklik)
- Contact (form sederhana)

## Tech Stack
- HTML, CSS, Vanilla JavaScript
- Google Fonts (Quicksand)
- Font Awesome (icon)
- Boxicons (icon tema)

## Fitur Utama
- **Responsive Navbar + Hamburger Menu**
  - Menu akan berubah menjadi hamburger di layar kecil dan dapat dibuka/tutup.
- **Dark Mode Toggle**
  - Klik tombol tema (moon/sun) untuk mengubah tema.
- **Skills Flip Card**
  - Card skill bisa dibalik (flip) saat diklik untuk menampilkan deskripsi.
- **Contact Form (Sederhana)**
  - Tombol Send menampilkan alert jika field lengkap (tidak mengirim email sungguhan).
- **Social Links**
  - Tautan social media ada di sidebar kanan.


## Cara Menjalankan
1. Cukup Buka file `index.html` lalu klik kanan search "Open In Default Browser", 
    atau gunakan Live Server (VSCode).

## Catatan
- Tombol "Send" hanya validasi sederhana menggunakan `alert`, bukan integrasi email.
- Jika icon tidak muncul, pastikan koneksi internet aktif (karena icon memakai CDN).
